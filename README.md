# Edge_vision_system
Multi-camera edge AI system using YOLOv8, DeepLabV3, DeepStream &amp; TensorRT

A complete multi-camera edge AI vision system built using:
- YOLOv8 (object detection)
- DeepLabV3 (semantic segmentation)
- DeepSORT-style tracking
- Video analytics & counting
- Multi-camera support
- Image stitching
- NVIDIA DeepStream pipeline
- TensorRT optimization

 Features:
- Real-time object detection
- Semantic segmentation
- Object tracking with unique IDs
- Crowd counting
- Anomaly detection
- Multi-camera pipeline
- Camera stitching (panoramic view)
- ONNX export for deployment
- DeepStream-ready architecture
